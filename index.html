Welcome to Word Guesser API.
Additional information

Web server: https://github.com/951219/WGwebserver
Chrome extentsion: https://github.com/951219/WGCE
Flutter app: https://github.com/951219/WGapp


Endpoints:
POST /est - add a word
GET /est/:id - get word by id
PATCH /est/:id - modify a word
GET /words - get all EST words
DELETE /est/:id - delete a word by id



GET /getbyword/:word - get word by word from wordnik
GET /eng/random/:number - get number of random words from wordnik
GET /eng/random - get 1 random word from wordnik
POST /eng - add a word
GET /eng/:id - get word by id
PATCH /eng/:id - modify a word
GET /eng - get all ENG words
DELETE /eng/:id - delete a word by id
