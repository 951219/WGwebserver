Welcome to Word Guesser API.
We have an app and a chrome extension in work

Endpoints:
GET /words - get all words
POST /words - add a word
GET /words/:id - get word by id in an array
PUT /words/:id - modify a word
DELETE /words/:id - delete a word by id
GET /words/random/:number - get number of random words
GET /wordsobject/:id - get a single object by id
GET /words/getbyword/:word - get word by word
GET /words/exists/:word - check if exists by word
GET /words/scrapefromeki/:word - scrape definition from EKI BROKEN
